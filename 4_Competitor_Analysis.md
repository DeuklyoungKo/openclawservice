# 경쟁 서비스 분석 (ClawBro vs Clawy.pro)

본 문서는 `1_PRD.md`에 정의된 OpenClaw AI 인프라 대행 서비스를 기준으로, 유사한 "OpenClaw / AI 인프라 매니지드 호스팅" 경쟁 서비스를 비교 분석합니다.

---

## 1. ClawBro (clawbro.ai/ko)
### 주요 서비스 모델 
- **설명:** 클릭 한 번으로 셀프 호스팅 웹 기반 OpenClaw(OpenSource AI) 배포를 제공하는 클라우드 호스팅 매니지드 서비스.
- **주요 기능:**
  - 24시간 라이브 상태 (개인 전용 격리 컨테이너 인스턴스)
  - 자동 업데이트, 보안 패치, 스케일링 일괄 수행 ("유지보수 없음" 강조)
  - 웹/앱 워크플로 자동화, 코드 도구, 파일 관리 등 다양한 오픈소스 AI 기능 제공
- **가격 정책 (Subscription 기준):**
  - **월 $29:** 2 vCPU, 4GB RAM, 40GB SSD
  - **월 $59:** 4 vCPU, 8GB RAM, 80GB SSD, 우선 지원
  - **월 $119:** 8 vCPU, 16GB RAM, 160GB SSD, 우선 지원
  - (공통: 매일 백업 및 프라이빗 컨테이너, 엔드투엔드 암호화)
- **강조하는 가치:** "서버를 찾고 설정하고 유지하는 장벽을 완전히 허물었다", "새로 배워야 할 기술 없음"

---

## 2. Clawy.pro (clawy.pro)
### 주요 서비스 모델
- **설명:** 텔레그램(Telegram) 기반으로 Claude, Kimi, MiniMax 등의 멀티 에이전트 AI 봇을 배포 및 연동해주는 No-coding 플랫폼.
- **주요 기능:**
  - **스마트 모델 라우팅:** 메시지 복잡도에 따라 최적의 모델로 자동 분배하여 API 비용 70% 최적화 달성.
  - **멀티 에이전트 오케스트레이션:** 한 봇에서 리서처, 코더 등 최대 8개의 전문 에이전트 동시 운용 가능.
  - **Memory & Storage:** 대화 문맥 및 장기 지식을 기억하는 3-Tier 메모리 시스템 적용.
  - **88개 빌트인 스킬:** Google Ads, Notion, Slack 등 사전 연동 기능 제공 (API Wiring 불필요).
  - **보안/프라이버시(데이터 제로 보관):** 사용자 메시지 보관 안 함, AES-256-GCM 암호화 등 프라이버시 유지.
- **가격 정책:**
  - **BYOK 플랜 ($7.99/월 예상 ~):** 개인 API 키 사용 (Anthropic 등), 16GB 제공, API 마진 없음, Brave 서치 200회 포함
  - **Pro 플랜 ($~):** API 기반 자체 요금제, 월 $15 플랫폼 크레딧, Agent Email 등 포함
  - **Pro+ 플랜 ($~):** 월 $100 플랫폼 크레딧, 추가 서치 및 우선 지원
- **강조하는 가치:** "단순히 묻고 답하는 챗봇이 아니라 일하는 에이전트 (A chatbot answers. An agent works.)"

## 3. 우리 서비스 (OpenClaw 대행 서비스)와의 차별성 비교

우리의 `1_PRD.md`에 따르면, 우리는 고객 소유의 AWS/GCP 서버를 직접 구성하고, 보안을 하드닝한 후 워크플로우를 연동해 주는 **프라이빗 인프라 설치 및 유지보수 컨설팅 대행**입니다.

| 비교 항목 | ClawBro | Clawy.pro | **우리의 서비스 (1_PRD.md)** |
|:---|:---|:---|:---|
| **플랫폼 형태** | 매니지드 클라우드 호스팅 (월정액) | 텔레그램 봇 기반 멀티 에이전트 플랫폼 | **고객 소유의 VPS(AWS/GCP) 대행 구축 및 관리** |
| **인프라 소유권** | 제공사 서버(격리 컨테이너) | 제공사 기반(Kubernetes Pod) | **고객이 100% 소유 (AWS/GCP 계정)** |
| **데이터 주권 및 보안** | 암호화 지원 (그러나 서비스 의존적) | 메시지 무보관 원칙 적용 | **100% 데이터 고객 통제 및 소유** (완전 격리) |
| **비즈니스 모델 / <br> 요금 구조** | 클라우드 구독형 <br>(월 $29 ~ $119) | 구독형 + 크레딧 <br>(월 $7.99 + 크레딧) | **일회성 셋업 비용 (50~150만원)** + <br>**월 유지보수 비용 (15~30만원)** + <br>**맞춤 컨설팅(100~500만원)** |
| **타겟 고객** | OpenClaw를 직접 운영하기 부담스러운 파워 유저 / 개인 창업자 | 워크플로 기능을 메신저(텔레그램)로 즉시 실행하고 싶은 유저 | **데이터 보안과 자사 고유 워크플로가 필수인 중소규모 사업자/에이전시** |
| **핵심 차별점** | 유지보수 Zero 강조, 빠른 배포 | 텔레그램 연동, 모델 라우터로 비용 절감 최적화 | **"내 서버" 기반 완전한 프라이빗 환경 구축 및 업무 맞춤형 컨설팅 연동 (Lock-in)** |

## 4. 벤치마킹 및 PRD(비즈니스 모델) 개선 인사이트

1. **"비용 절감" 가치 제공의 고도화 (Clawy.pro 벤치마킹)** 
   - `1_PRD.md`의 Tier 2 (매니지드 서비스)에서 제공하는 "API 비용 최적화 리포트" 기능은 훌륭한 훅(hook)입니다. Clawy.pro는 "스마트 모델 라우팅으로 최대 70% 비용 절감" 메시지로 어필하고 있습니다.
   - **적용점:** 복잡하지 않은 작업에는 가벼운 모델을, 전문적인 코딩이나 리서치에는 무거운 모델을 배분하여 체감 비용을 줄여주는 기능을 OpenClaw 셋업에 추가할 수 있음을 어필해야 합니다.

2. **메신저(Telegram/Slack) 지원의 필요성 (Clawy.pro 벤치마킹)** 
   - OpenClaw를 컴퓨터 웹 브라우저가 아니라, 이동 중에도 언제든 실행 가능한 **텔레그램이나 슬랙 봇 형태**로 모바일에서 연동 시키는 것은 강력한 편의성을 줍니다.
   - **적용점:** Tier 3(컨설팅)에 "Slack/Telegram 프라이빗 봇 연동 시스템" 구축 항목을 추가하여 상품 가치를 올려 판매할 수 있습니다. 이미 Clawy.pro가 이 니즈를 대행하여 시장을 모으고 있습니다.

3. **데이터 주권 및 비용 체감 어필 (ClawBro 벤치마킹)**
   - ClawBro는 클라우드 호스팅 기반이라 컴퓨팅 파워(vCPU/RAM)에 따라 비용이 수직 상승(월 최대 $119)합니다. 
   - **적용점:** 우리의 서비스는 AWS/GCP의 원가(원하는 만큼 리소스 확장)만 지불하면서 우리에게는 고정 매니지드 비용(월 15~30만원)만 내는 형태임을 강조해 **"사업 확장 시 서버 규모가 커질수록 직접 호스팅이 저렴하다"**는 프레임을 세워야 타당성을 증명할 수 있습니다. 더불어 "내 서버"가 가지는 궁극의 BYOK(데이터 주권) 철학을 세일즈 키워드로 가져가야 합니다.
