# OpenClaw AI 운영 대행 서비스
### 비즈니스 모델 기획서 v1.0

---

## 1. 비즈니스 개요

AI 에이전트 도입을 원하는 **중소규모 사업자 및 1인 창업자**를 대상으로, 보안이 강화된 OpenClaw 환경을 구축하고 지속적으로 관리해주는 **풀스택 AI 인프라 대행 서비스**입니다.

> **OpenClaw란?**
> Claude API 기반의 AI 에이전트 플랫폼으로, 자체 서버 위에서 운영되는 오픈소스 에이전트 실행 환경입니다.
> 고객은 SaaS형 AI 툴 대신 **자신의 인프라 위에서** AI 에이전트를 운영할 수 있습니다.

---

## 2. 타겟 고객 및 시장 문제

### 타겟 고객
- 쇼핑몰, 에이전시, 프리랜서 등 **월 매출 500만~5,000만원 규모** 1인~10인 사업자
- AI 도구에 관심은 있지만 **서버 세팅, 보안, 유지관리**를 직접 하기 어려운 비개발자 창업자
- ChatGPT/Claude 등 SaaS 툴의 **데이터 유출 리스크**를 우려하는 고객
- 맥미니 구입에 부담이 있는 고객

### 현재 고객의 대안 (경쟁 상황)
| 현재 대안 | 문제점 |
|---|---|
| SaaS AI 툴 직접 사용 | 데이터 통제 불가, 비용 예측 어려움 |
| IT 프리랜서 단발 의뢰 | 유지관리 공백, 보안 사후관리 없음 |
| AWS 공식 파트너사 | 중소 규모엔 과도한 비용과 복잡성 |
| 직접 구축 시도 | 시간·기술 부족으로 포기율 높음 |

> **핵심 인사이트:** 고객이 원하는 건 "AI 에이전트"가 아니라 **"직접 안 해도 되는 AI 업무 자동화"** 입니다.

## 3. 서비스 티어 (Tiered Service Model)

### Tier 1 — 인프라 구축 대행 (Setup Fee)

**"처음 한 번, 제대로 세팅"**

- GCP / AWS 계정 생성 가이드 및 초기 설정 지원
- VPS Hardened 보안 프로필 기본 적용 (방화벽, SSH 키 인증, 포트 제한)
- OpenClaw 설치 및 초기 에이전트 구성
- BYOK(Bring Your Own Key) 설정: API 키는 고객이 직접 보관

**예상 가격:** 50만~150만원 (규모 및 요구사항에 따라 차등)
**수익 모형:** 일회성 프로젝트 수수료

---

### Tier 2 — 매니지드 서비스 (Monthly Subscription)

**"운영 걱정 없이, 본업에 집중"**

- 월간 보안 패치 및 OpenClaw 버전 업데이트
- 실시간 로그 모니터링 및 이상 징후 알림
- **AI API 비용 최적화 리포트** (월 1회, 고객 절감액 가시화) ← 핵심 훅
- 장애 발생 시 4시간 내 대응 SLA

**예상 가격:** 월 15만~30만원
**수익 모형:** 반복 구독 수익 (MRR 기반)

> **Tier 2 포지셔닝 전략:** "보안 업데이트"보다 **"이번 달 API 비용 23% 절감했습니다"** 라는 리포트가 고객 체감 가치를 극대화합니다. 비용 절감 수치를 전면에 내세우세요.

---

### Tier 3 — 비즈니스 로직 연동 (Consulting)

**"내 업무 흐름에 맞는 AI 자동화"**

- 맞춤형 워크플로우 설계 및 이식
  - 웹 스크래핑 자동화 (경쟁사 가격 모니터링, 리드 수집 등)
  - 결제·주문 자동화 파이프라인
  - Notion / Slack / 구글 시트 연동
  - 고객사 전용 AI 에이전트 커스터마이징
- 워크플로우 인수인계 문서 제공

**예상 가격:** 프로젝트당 100만~500만원
**수익 모형:** 컨설팅 프로젝트 수수료

> **Tier 3의 전략적 역할:** 고객 업무 시스템에 깊이 연동될수록 **자연스러운 이탈 방지(Lock-in)** 효과가 발생합니다. 단순 구독보다 훨씬 강력한 고객 유지 장치입니다.

## 4. 수익 구조 요약

```
[ 고객 유입 ]
      ↓
[ Tier 1: Setup ] — 일회성 수익 + 신뢰 확보
      ↓
[ Tier 2: 구독 ] — 안정적 MRR 확보
      ↓
[ Tier 3: 컨설팅 ] — 고단가 프로젝트 + Lock-in
```

| 티어 | 유형 | 예상 단가 | 목표 고객 수 | 월 예상 수익 |
|---|---|---|---|---|
| Tier 1 | 일회성 | 100만원 | 월 3건 | 300만원 |
| Tier 2 | 구독 | 20만원/월 | 20명 | 400만원 |
| Tier 3 | 프로젝트 | 200만원 | 월 1건 | 200만원 |
| **합계** | | | | **약 900만원/월** |

## 5. 운영 원칙 및 보안 전략

### BYOK (Bring Your Own Key) 원칙
- 고객의 Anthropic / OpenAI API 키는 **고객 본인이 직접 생성·보관**
- 서비스 제공자(OpenClaw 대행사)는 API 키에 접근하지 않음
- **법적·보안적 책임 완전 분리** → 고객 신뢰도 향상, 서비스 제공자 리스크 최소화

### VPS Hardened 기본 보안 프로필
- SSH 키 인증만 허용 (패스워드 로그인 차단)
- UFW 방화벽 + 최소 포트 개방 원칙
- Fail2ban 침입 시도 차단
- 정기 보안 감사 로그 보관

### 차별화 포인트
| 경쟁 요소 | 일반 AI SaaS | IT 프리랜서 | **OpenClaw 대행** |
|---|---|---|---|
| 데이터 주권 | ✗ 클라우드 서버 저장 | △ 케이스 바이 케이스 | ✅ 고객 서버 100% 통제 |
| 보안 유지관리 | △ 플랫폼 의존 | ✗ 납품 후 종료 | ✅ 지속적 모니터링 |
| 비용 예측 가능성 | ✗ 사용량 기반 변동 | ✗ 견적 불투명 | ✅ 구독 + 최적화 리포트 |
| 업무 맞춤화 | ✗ 범용 기능만 | △ 가능하나 단발성 | ✅ 지속 개선 파트너십 |
